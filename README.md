描述
墨水瓶天气气象站-ESP8266
前言
近期市面上出现了一批LG的WF42型号三色墨水屏驱动板，论坛中DUCK大佬已发布相关固件。不过其服务器基于PHP开发，且公众服务器需激活码才能使用。好在DUCK开源了全部原理图与源码，本项目以此为基础进行优化：将温湿度传感器更换为SHT40，并新增一颗三轴磁力计以替代传统按钮（该部分逻辑目前仍在完善中）。

烧录过程
烧录文件系统
打开路径“C:\Users\你的用户名.arduinoIDE”，在此目录下新建文件夹并命名为“plugins”。
访问“https://github.com/earlephilhower/arduino-littlefs-upload/releases”，下载最新版本的vsix文件。
将下载好的vsix文件复制到刚创建的“plugins”文件夹内。
重启Arduino IDE，按下F1键，在弹出的搜索框中输入“Upload”，选择“Upload LittleFS to Pico/ESP8266/ESP32”选项，完成文件系统的烧录操作。
烧录源码
启动Arduino IDE后，在开发板选择界面中，将开发板型号设置为ESP12F。
确认源码无误后，点击烧录按钮，等待烧录完成即可。
服务器搭建
本项目采用Python搭建简易接口服务器，并已将接口API修改为和风天气V7版本。
运行github仓库中的server.py文件前，需提前准备：
和风天气API及对应的令牌（需自行申请）。
高德地图接口API令牌（需自行申请）。
将上述申请到的令牌信息填入server.py文件的对应位置。
根据实际网络环境，修改源码中的IP地址和端口号，确保服务器能正常访问。
完成配置后，直接运行server.py即可启动服务器。
外壳装配
外壳可直接通过3D打印制作，打印参数可根据所使用的打印机型号进行调整，建议采用PLA材质，层厚设置为0.2mm，填充率20%即可。
前面板直接采用LG原装前面板，以保证与驱动板的适配性。
螺丝选用原装螺丝，不仅能降低装配成本，还能确保外壳各部件连接牢固，避免因尺寸不匹配导致的装配问题。
